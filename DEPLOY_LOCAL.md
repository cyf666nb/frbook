# frbook 部署指南

> 纯前端版本，无需安装数据库或后端服务！

## 环境要求

- Node.js 16+

## 部署步骤

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

### 3. 构建生产版本

```bash
npm run build
```

构建产物在 `dist/` 目录，可直接部署到静态服务器。

## 数据重置

如需重置所有数据（包括测试账号），在浏览器控制台执行：

```javascript
localStorage.clear()
window.location.reload()
```

## 常见问题

### Q: 数据保存在哪里？
A: 使用浏览器 localStorage，无需服务器。

### Q: 如何添加自定义数据？
A: 登录后使用"发布图书"功能添加。

### Q: 如何清除所有数据？
A: 执行 `localStorage.clear()` 后刷新页面。
